# GamesDB

This application allows users to search for video games using API from https://thegamesdb.net/. 

---

# Features

1. Search for games by title
2. View multiple search results
3. Display game details
4. Browse games by platform
5. Responsive Bootstrap 5 design
6. Shared PHP templates (header, navigation, footer)
7. API integration

---
# Technologies Used

1. PHP
2. HTML5
3. CSS
4. Bootstrap 5
5.  JSON
6.  API

---

# Setup Instructions

1. Clone or download the repository.
2. Find and add API key to config.php
3. Replace `YOUR_API_KEY_HERE`.
4. Place the project inside your web server directory (XAMPP htdocs).
5. Start Apache and open the project in your browser.
